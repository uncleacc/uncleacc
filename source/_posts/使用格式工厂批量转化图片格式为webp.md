## 下载格式工厂
这一步我就不放链接了，可以百度搜索格式工厂官方下载
## 使用
打开下载好的格式工厂  
{% fb_img https://cdn.jsdelivr.net/gh/uncleacc/Sucai/0.png %}  
点击转化为webp👈（也可以转换成其他格式，操作流程类似）    
{% fb_img https://cdn.jsdelivr.net/gh/uncleacc/Sucai/1.png %}    
先选好你想储存的文件夹👈  
{% fb_img https://cdn.jsdelivr.net/gh/uncleacc/Sucai/2.png %}  
添加图片👈      
{% fb_img https://cdn.jsdelivr.net/gh/uncleacc/Sucai/3.png %}   
点击确定👈  
{% fb_img https://cdn.jsdelivr.net/gh/uncleacc/Sucai/4.png %}
完成👈   
然后图片就保存到你想储存的文件夹里了，而且名称没有任何变化只是前缀变了而已，更重要的是转换一百多张图片只需要一分钟左右，良心软件😆   
然后就可以用图片做你想做的事了
